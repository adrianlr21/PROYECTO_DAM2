📱 PROYECTO_DAM2 – Aplicación iOS con Core Data

📌 Descripción

- Proyecto grupal desarrollado en Xcode como parte del curso DAM, enfocado en la creación de una aplicación móvil para iOS.
- La aplicación implementa almacenamiento local de datos utilizando Core Data, permitiendo la gestión persistente de información dentro del dispositivo.

🛠 Tecnologías utilizadas

- Lenguaje: Swift
- Entorno de desarrollo: Xcode
- Persistencia de datos: Core Data
- Control de versiones: Git & GitHub

🧩 Funcionalidades principales

- Interfaz gráfica desarrollada en Xcode
- Navegación entre pantallas
- Almacenamiento y recuperación de datos con Core Data
- Gestión de registros dentro de la aplicación
- Interacción dinámica con el usuario

🧠 Implementación técnica

- Uso de Core Data para persistencia local
- Creación de entidades y modelo de datos
- Operaciones CRUD (crear, leer, actualizar, eliminar)
- Manejo del contexto (NSManagedObjectContext)
- Organización del proyecto en capas
